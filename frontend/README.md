## Build

## Futasd a develpoment szervert:

```bash/terminal
cd frontend

npm install (letölti az össszes csomagot ami a package.json fájlban leírtunk, szükséges hogy fusson a program)

npm run dev

Másoldki az url-t és másold be egy keresőbe.(Az első linket másold ki)
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. (Valahogy így nézki az URL)
